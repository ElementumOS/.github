# Japanese
## ここはElementumOSのGitHub組織です！
ElementumOSは日常利用に重点を置いたBasix Debian Edition派生のディストリビューションです。

### 特徴
* 通常のLinuxカーネルの代わりにLiquorix Kernelと呼ばれるZen Kernel派生のカスタムカーネルを利用
* カスタマイズ性の高いXfce4とLightDM (web-greeter)を採用
